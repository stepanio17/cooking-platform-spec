# Платформа для кулинарных рецептов



Веб-приложение для поиска, создания и хранения кулинарных рецептов. Проект позволяет пользователям делиться своими кулинарными идеями, создавать личные подборки и искать блюда по содержимому холодильника.



### Ключевые возможности



* **Умный поиск рецептов**: Фильтрация по категориям, поиск по названию, а также продвинутый поиск с включением или исключением конкретных ингредиентов.
* **Динамический перерасчет порций**: Ингредиенты автоматически пересчитываются прямо в карточке рецепта при изменении количества порций.
* **Пошаговые инструкции**: Поддержка создания рецептов с детальными шагами приготовления и фотографией для каждого шага.
* **Социальные функции и рейтинги**: Система оценок "Рекомендую / Не рекомендую", счетчик просмотров и добавление в "Избранное".
* **Личные коллекции**: Возможность создавать собственные папки и сохранять туда рецепты.



### Как запустить проект



Проект разделён на два репозитория и полностью упакован в Docker-контейнеры. Для запуска потребуется установленный Docket и Git.


1. **Создайте в git корневую папку для проекта и перейдите в неё:**

mkdir recipe-platform
cd recipe-platform

2. **Склонируйте серверную и клиентскую части в созданную папку (обязательно используйте названия папок backend и frontend):**

git clone https://github.com/stepanio17/cooking-platform-frontend.git frontend
git clone https://github.com/stepanio17/cooking-platform-backend backend

3. **Перейдите в папку бэкенда, где находится конфигурационный файл, и запустите сборку:**

cd backend
docker compose up --build

4. **После завершения сборки проект будет доступен по адресу:**



http://localhost:5173
