Спецификация к “Платформе для кулинарных рецептов”
Цель
Создать веб-приложение для выбора рецептов по потребностям пользователя.
Целевая аудитория
Домохозяйки/домохозяины, непрофессиональные повары.
User stories
Как человек, у которого мало времени, я хочу находить быстрые рецепты, чтобы я мог потратить больше времени на отдых.
Как студент, у которого мало денег, я хочу выбирать недорогие рецепты, чтобы экономить деньги.
Как начинающий повар, я хочу выкладывать свои рецепты, чтобы проверить качество своих рецептов.
Как веган, я хочу убирать продукты животного происхождения, чтобы следовать своей идеологии.
Функции
У пользователя: заходить в личный кабинет, сохранять рецепты, добавлять новые рецепты, создавать коллекции из сохранённых рецептов.
У рецепта: количество просмотров рецепта, ставить оценку рецепту, перерасчёт ингредиентов от количества порций
У поиска рецептов: искать по желаемым ингредиентам, исключать ненужные ингредиенты, ограничить рецепты для веганов, вегетарианцев, людей с аллергией на глютен и лактозу.
У личного кабинета: регистрация нового пользователя, вход старых пользователей, защита персональных данных, восстановление пароля.
У модератора: одобрять предложенный пользователем рецепт
Требования
Приложение должно отображаться
Технологии
Frontend: React
Backend: Python
База данных: PostgreSQL
