# **Спецификация текущего релиза (Версия 1.0)**



### **1. Статус проекта**



Данная версия является Proof of Concept (PoC) и представляет собой первоначальную наработку платформы для кулинарных рецептов.



### **2. Реализованный функционал**



#### **Рецепты**



* **Просмотр списка:** Автоматическое подтягивание всех существующих рецептов из базы данных PostgreSQL на главную страницу.
* **Добавление рецепта:** Форма связи, позволяющая отправить данные на сервер.
* **Поля данных:** 

  * Название рецепта (title). 
  * Описание рецепта (description). 
  * Количество порций (servings). 
  * ID автора (author\_id) для внутренней идентификации.



### **3. Ограничения**



* Поиск и фильтрация по ингредиентам пока не реализованы.
* Система авторизации и личных кабинетов заменена на привязку author\_id к 1.
* Загрузка мультимедиа временно ограничена текстовыми описаниями.
